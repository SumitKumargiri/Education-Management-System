A modern and scalable **Education Management System** built using **Angular 17, TypeScript, Tailwind CSS, and Angular Material**.

This application helps manage educational operations such as students, teachers, attendance, meetings, dashboards, analytics, scheduling, and more with a responsive and professional UI.

---

## 🚀 Features

* 👨‍🎓 Student Management
* 👨‍🏫 Teacher Management
* 📅 Meeting & Scheduling System
* 🏫 Classroom & Room Management
* 📊 Dashboard & Analytics
* 📈 Charts & Reports
* 🔐 Authentication & JWT Support
* 📱 Responsive UI Design
* 🌙 Modern Angular Material Components
* ⚡ High Performance Angular SPA
* 🔔 Toast Notifications & Alerts
* 📂 SSR Support (Angular Universal)

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/SumitKumargiri/Education-Management-System.git
```

---

### 2️⃣ Navigate to Project

```bash
cd Education-Management-System
```

---

### 3️⃣ Install Dependencies

```bash
npm install --legacy-peer-deps
```

---

### 4️⃣ Run Application

```bash
ng serve
```

Open browser:

```bash
http://localhost:4200
```
